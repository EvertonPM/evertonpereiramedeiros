Portifólio

Ifood: Para um projeto acadêmico na Faculdade de Análise e Desenvolvimento de Sistemas, desenvolvi uma réplica do aplicativo iFood. Este projeto envolveu a criação de uma plataforma que replica funcionalidades essenciais do iFood, como seleção de restaurantes, realização de pedidos e acompanhamento de entregas. Através dessa experiência, pude aprimorar minhas habilidades de desenvolvimento de software e compreender melhor os desafios e processos envolvidos na construção de aplicativos semelhantes..

Machine Lerning: Desenvolvi um modelo de machine learning capaz de classificar vinhos entre tintos e suaves. Utilizando técnicas de processamento de dados e algoritmos de aprendizado supervisionado, o modelo foi treinado com uma ampla variedade de características sensoriais dos vinhos, como teor alcoólico, acidez, doçura e taninos. Essa experiência me permitiu aprofundar meu conhecimento em análise de dados e machine learning, além de explorar a aplicação prática dessas técnicas no campo da enologia.
